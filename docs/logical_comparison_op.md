# Supported Comparison operators

| Description           | CQL2 Operator(s) | Resulting OData Operator |
|-----------------------|------------------|--------------------------|
| Equal                 | `=`, `eq`        | `eq`                     |
| Greater than          | `>`,`gt`         | `gt`                     |
| Greater than or equal | `>=`,`gte`       | `ge`                     |
| Less than             | `<`, `lt`        | `lt`                     |
| Less than or equal    | `<=`, `lte`      | `le`                     |
| Is a member of        | `in`             | `in`                     |

# Supported Logical operators

| Description | CQL2 Operator(s) | Resulting OData Operator |
|-------------|------------------|--------------------------|
| Logical and | `and`            | `and`                    |
| Logical or  | `or`             | `or`                     |
